# 🚀 QuestionForge – AI Question Paper Generator

QuestionForge is a modern AI-powered Question Paper Generator that creates professional examination papers from uploaded syllabus files or pasted notes. It works entirely in the browser without requiring any backend or API, making it fast, private, and easy to use.

## ✨ Features

- 📄 Upload PDF, TXT, HTML, CSV, JSON, and Markdown files
- 🤖 Automatically generates question papers from syllabus content
- 📝 Supports multiple question types:
  - Multiple Choice Questions (MCQs)
  - True/False
  - Fill in the Blanks
  - Short Answer Questions
  - Long Answer Questions
  - Case Study Questions
- 🎯 Adjustable difficulty levels (Easy, Mixed, Hard)
- 📊 Custom marks distribution
- 📚 Custom subject, class, duration, and exam title
- 👀 Toggle Answer Key
- 🖨️ Print-ready question papers
- 📥 Export as HTML and TXT
- 🔒 100% Offline (No API Required)

## 🎯 How It Works

1. Upload syllabus files or paste notes.
2. Configure exam settings.
3. Select question types and marks.
4. Choose difficulty level.
5. Click **Generate Question Paper**.
6. Print or download the generated paper.
